# Perl Scripts Collection

A toolbox of functional Perl scripts

## Contents

### [File Management](./file-management)
* `file_exists.pl`: Checks if file exists
* `read_files.pl`: Reads a file, filters content and prints out line by line
* `write_files.pl`: Reads from an input file, performs text replacement and prints to an output file

### [Text Processing](./text-processing)
* `capture_groups.pl`: Utilizes Regular Expressions to extract specific data patterns

### [Web Automation](./web-automation)
* `download_image.pl`: Interacts with external web services to retrieve media assets
